# Getting started
- Ensure that you have node installed on your computer
- Clone the repo to your machine
- Start the server:
    ```
    cd server
    node index.js
    ```
- Start the client:
  ```
  cd client
  npm start
  ```
- From here you'll be able to see the posts in the tree menu and click posts to edit the location and author fields
